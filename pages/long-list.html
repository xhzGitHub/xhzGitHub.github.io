          # 什么是长列表

          在讨论虚拟列表之前，我们先来看下什么是"长列表"。

          所谓的长列表，顾名思义，类似于我们平常刷手机 app 时见到的瀑布流，无限加载列表等。在无限下滑的过程中，Dom 中的节点数也在无限增加，这可能会造成客户端内存消耗高、加载卡顿等情况的出现。其实随着 web 引擎性能的提高，很少会见到组件库（类似 infinite scroll）中有类似的长列表优化的操作，但我认为在部分场景下这样的优化还是大有用武之地的。

          # 什么是虚拟列表

          其实优化无限加载导致 Dom 节点数疯狂增加的思路很简单：

          保证渲染中的 Dom 节点数在固定数量即可，在页面滚动的过程，实时更新被渲染的元素。

          虚拟列表就是形容以上思路的一套操作，它不是真实所有数据渲染出得列表，而是通过计算得到的固定 Dom 数的列表。

          # 实现起来

          核心思路：渲染首屏时，即 startIndex 为0，假设我们想看到10个元素，那我们的 visibleData 中就应该放 15个元素，来让容器可以滑动。那么当首屏时，可视区域一屏可以渲染10个，我们看到的是 item1 ~ item10。

          当用户向下滑动时，记录下当前滑动的距离 scrollTop，找到当前屏幕中可见的第一个 item（我们称其为锚点元素 anchorItem），当用户划过了一个元素，则 item1 成为了锚点元素。然后重新计算 startIndex，通过 startIndex 与 endIndex 来获取当前渲染的数据数组，visibleData = data.slice(startIndex, endIndex)。

          第一步结束后，我们就已经实现了保持 Dom 数不变，随着屏幕的 scroll 不停地更新 Dom 中的内容的目标。

          但是我们发现，当我们滚动到 item15，就没办法往下滑动了，是因为我们的列表容器本身就那么长。那如果我们随着屏幕的滚动，实时增加列表的长度呢？

          我们可以使用一个巧妙的方法，就是每当屏幕划过一个 item 时，我们就给外层的列表容器重新计算下 paddingTop，paddingTop 的高度等于划过 n 个 item 的高度之和。

          源码奉上：

          // App.js

          import './index.less'

import React, { Component } from 'react'
import ItemTest from '../src/components/long-list/ItemTest'

const height = 60
const bufferSize = 5

export default class VirtualizedList extends Component {
  constructor(props) {
    super(props);

    this.state = {
      startOffset: 0,
      endOffset: 0,
      visibleData: []
    };

    this.data = new Array(50).fill(0);

    for(let i = 0; i < 50; i++) {
      this.data[i] = i;
    }

    this.startIndex = 0;
    this.endIndex = 0;
    this.scrollTop = 0;

    this.doc = null;

    // 缓存已渲染元素的位置信息
    this.cache = [];
    // 缓存锚点元素的位置信息
    this.anchorItem = {
      index: 0, // 锚点元素的索引值
      top: 0, // 锚点元素的顶部距离第一个元素的顶部的偏移量(即 startOffset)
      bottom: 0 // 锚点元素的底部距离第一个元素的顶部的偏移量
    };

    this.handleScroll = this.handleScroll.bind(this);
    this.cachePosition = this.cachePosition.bind(this);
  }

  handleScroll(e) {
    if (!this.doc) {
      // 兼容 iOS Safari/Webview
      this.doc = window.document.body.scrollTop ? window.document.body : window.document.documentElement
    }

    const scrollTop = this.doc.scrollTop
    if (scrollTop > this.scrollTop) {
      if (scrollTop > this.anchorItem.bottom) {
        this.updateBoundaryIndex(scrollTop);
        this.updateVisibleData();
      }
    }
    this.scrollTop = scrollTop;
  }

  cachePosition(node, index) {
    const rect = node.getBoundingClientRect();
    const top = rect.top + window.pageYOffset;
    
    this.cache.push({
      index,
      top, // 元素顶边距离 window 顶部的距离
      bottom: top + height // 元素底边距离 window 顶部的距离
    });

    console.log('cache:', this.cache);
  }

  updateBoundaryIndex(scrollTop) {
    scrollTop = scrollTop || 0;

    const anchorItem = this.cache.find(item => item.bottom >= scrollTop);

    console.log('anchorItem:', anchorItem);
    this.anchorItem = { ...anchorItem };
    this.startIndex = this.anchorItem.index;
    this.endIndex = this.startIndex + this.visibleCount;
  }

  updateVisibleData() {
    const visibleData = this.data.slice(this.startIndex, this.endIndex);
    console.log('visibleData:', visibleData);
    this.setState({
      ...this.state,
      startOffset: this.anchorItem.top,
      visibleData
    })
  }

  componentDidMount() {
    this.visibleCount = Math.ceil(window.innerHeight / height) + bufferSize;
    this.endIndex = this.startIndex + this.visibleCount;
    this.updateVisibleData();

    window.addEventListener('scroll', this.handleScroll, false);
  }

  componentWillUnmount() {
    if (this.handleScroll) {
      window.removeEventListener('scroll');
    }
  }

  render() {
    const { startOffset, endOffset, visibleData } = this.state;

    return (
      <div className='wrapper'>
        <div style={{ paddingTop: `${startOffset}px`, paddingBottom: `${endOffset}px` }}>
          {
            visibleData.map((item, index) => {
              return (
                <ItemTest
                  cachePosition={this.cachePosition}
                  key={this.startIndex + index}
                  index={this.startIndex + index}
                />
              )
            })
          }
        </div>
      </div>
    );
  }
}
